#   Aplikacja Shiny do ściagania danych KRS

   Oprogramowanie do ściągania danych **KRS** ze strony **Ministerstwa Sprawiedliwości** wykorzystujące dostarczane pliki w formacie **.json** udostepniane przez ministrstwo.

Aplikacja jest dostępna pod adresem: [API KRS Ministerswo Sprawiedliwości](https://adamatusz.shinyapps.io/MS_KRS/)
